How to update your live site

One-file swap, no coding needed.

  1. Click Download content.js below
  2. Go to your GitHub repository
  3. Find content.js → click the pencil ✏️ icon
  4. Select all, delete, paste your downloaded file's contents
  5. Click Commit changes
  6. Live in ~30 seconds ✅
